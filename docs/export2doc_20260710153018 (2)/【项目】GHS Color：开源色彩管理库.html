¶前言

你是否也经常苦恼于找不到合适的颜色？是否苦恼于没有办法统一管理自己的设计色彩？

那么，我的新项目，GHS Color或许能解决你的问题！

¶项目介绍

GHS Color是一个由Garbage Human Studio开发的开源色彩管理器，旨在为设计师、开发者和任何需要颜色灵感的人提供一个便捷的平台。它不仅拥有丰富的颜色库，还支持多种颜色格式复制、收藏夹功能，让你轻松管理和使用你心仪的颜色。🎉

¶功能介绍

GHS Color遵循简洁、高效、美观的原则进行设计。

你可以通过以下永久链接访问🚀：

https://color.ghs.red

其主要优势在于：

✅ 丰富可自定义的颜色库： 你可以随意在js代码中添加你的颜色，进行统一管理。🌈

✅ 多种颜色格式支持： 只需添加HEX格式，即可一键轻松复制HEX、RGB、HSL格式到各种软件中使用。🎨

✅ 收藏夹功能： 一键收藏你喜欢的颜色。 ❤️

✅ 主题和语言切换： 亮色、暗色主题切换，中英文多语言支持。 🌐

✅ 流畅的动画效果： 让你的使用体验更加愉悦。 ✨

¶未来方向

GHS Color是一个完全开源的项目，你可以在以下链接找到它的源码：

Mystic-Stars/GHS-Color (github.com)

你可以根据自己的需求进行修改和定制。💡

GHS Color仍在不断完善中，我们需要并接受更多颜色库。

如果你有好的颜色主意，可用以下格式在Github中提Pull request： 

    {
        name: "Box Yellow",
        nameZh: "盒子黄",
        hex: "#f6dc50",
        description: "The exclusive yellow color of BoxWorld, the logo color of GHS.",
        descriptionZh: "盒王的专属黄色，GHS的标志颜色。",
        temperature: "warm"
    }
