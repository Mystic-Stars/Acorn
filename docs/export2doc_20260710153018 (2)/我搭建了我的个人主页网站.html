项目介绍

这是我的个人主页网站项目，基于Codefreyy的joy-personal-portfolio项目进行开发。你可以在本项目基础上随意进行修改，将其修改为你的个人主页并发布。

如果喜欢此项目，请给此仓库Star。

开发指南

入门引导

这是一个使用 create-next-app 引导程序创建的 Next.js 项目。

如果你要对其进行二次开发，首先，运行开发服务器：（确保已安装 Next.js ） 

npm run dev
# or
yarn dev
# or
pnpm dev

大功告成后，你可以在在浏览器中打开 http://localhost:3000 查看页面。

你可以通过修改文件来开始编辑页面。当你编辑文件时，页面会自动更新并实时修改。

该项目使用 next/font 自动优化并加载 Inter 字体，这是一种自定义的 Google 字体。

在 Vercel 上部署

部署 Next.js 应用最简单的方式是使用 Next.js 创建者提供的 Vercel 平台。

所以我推荐您在Vercel上部署此项目。
