前言

伴随着 Cursor 新计费方式的推出，降智在AI IDE中越来越常见，以至于让 Cursor 有时蠢到一种几乎不可用的地步。在此情况下，Augment 便是替代 Cursor 的一个绝佳选择。其作为插件拥有更高的兼容性，并且独一无二的 AugmentContextEngine 使其具有竞争力，在许多场景效果超过 Cursor 等竞品。但是其每月50美元的价格确实太高了，好在我们可以通过试用会员进行轻度使用的白嫖。

详细步骤

  1. 安装poolhub提供的已修补的 去💩版 Augment VSCode 插件，在其中重置机器码（注意：这会移除几乎所有的Augment本地数据，包括Augment的项目记忆等

  2. 在 VSCode 中设置git信息

    git config --local user.email “注册augment时用的邮箱”
git config --local user.name “邮箱@前的部分”

    由于Augment在初始化时会读取本地git信息，所以这一步可以大大提高账号存活概率

  3. 在浏览器中新建“无痕浏览窗口”，在其中注册Augment，注意使用第二步git信息中的邮箱地址。 如果有条件可以直接使用指纹浏览器！
