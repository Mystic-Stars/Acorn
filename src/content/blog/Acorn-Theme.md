## 写在前面

折腾博客这件事，大概永远不会有真正的“结束”。从最初想找一个能安静写点东西的地方，到后来慢慢加上归档、友链、评论、搜索、统计……功能越来越多，也越来越想把它做成真正属于自己的样子。

所以，Acorn 就这样诞生啦 🌰

它是我为这个博客制作的一套 Astro 主题。整体参考了动森那种温暖、轻松的小岛氛围，但没有把内容藏在花里胡哨的效果后面——我还是希望打开网站的人，能舒服地阅读文章、找到内容，也能看到这个博客一点一点长大。

## Acorn 有什么？

目前它已经包含了我自己日常会用到的一些功能：

- 首页文章流、置顶文章与分页
- 分类、标签和归档筛选
- 全站搜索、RSS、Sitemap 与 robots.txt
- Markdown 文章目录、代码复制、图片预览和表格优化
- 友链小岛与 GitHub PR 提交流程
- Twikoo 评论、访问统计与留言管理页
- 移动端适配，以及一套可以自己继续调的主题颜色配置

这些功能不一定都是“必须”的，但对于一个想长期维护的博客来说，能少一点重复折腾，多一点安心写作，总是好事。

## 不只是给我自己用

这次我也把 Acorn 单独整理成了公开模板仓库：

- 主题仓库：[Mystic-Stars/acorn-theme](https://github.com/Mystic-Stars/acorn-theme)
- 在线演示与使用手册：[acorn.mysticstars.cn](https://acorn.mysticstars.cn)

如果你也想搭一个类似风格的博客，可以直接在 GitHub 点击 **Use this template** 创建自己的仓库。模板站里已经准备好了从创建项目、修改站点资料、写文章，到部署 Vercel、配置评论和统计的完整说明。

我不希望 Acorn 只是“拿来看看”的展示项目。更希望它能成为一个拿到手就能慢慢改、慢慢写的起点。

## 接下来

目前 Acorn 还在持续完善中。后面大概还会继续优化文章阅读体验、配置文档和一些容易被忽略的小细节；如果你用了它，也欢迎提 Issue、提 PR，或者单纯来告诉我哪里不好用。

毕竟博客就像一座小岛——不用一下子建得多么宏大，但可以慢慢种树、铺路、放上喜欢的东西。

希望 Acorn 能陪我把这座岛继续经营下去，也希望它能帮到同样喜欢折腾博客的你。✨
